## Introduction

Olyptik is a powerful web scraping and data extraction platform that helps you gather and process web data efficiently. This guide will walk you through the essential steps to get started with our API.

## Getting Started with the API

### Authentication

First, you'll need an API key. You can get one from your [settings page](https://app.olyptik.io/settings/project). Include it in your requests using the Authorization header:

```bash
Authorization: Bearer your-api-key
```

### Step 1: Create a Crawl

Start by creating a crawl with your target URL. This initiates the web scraping process:

<CodeGroup>

```javascript javascript
// npm i olyptik
import Olyptik from "olyptik";

const olyptik = new Olyptik({ apiKey: 'your-api-key' });

const crawl = await olyptik.runCrawl({
  startUrl: 'https://example.com',
  maxResults: 50
});
  
console.log('Crawl created:', crawl.id);
```


```python python
# pip install olyptik
from olyptik import Olyptik

# Initialize with API key
client = Olyptik(api_key="your_api_key_here")

# Start a crawl
crawl = client.run_crawl({
    "startUrl": "https://example.com",
    "maxResults": 50
})

print(f"Crawl started with ID: {crawl.id}")
```


```bash curl
curl -X POST https://api.olyptik.io/crawls \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "startUrl": "https://example.com",
    "maxResults": 50
  }'
```

</CodeGroup>

### Step 2: Monitor Crawl Status

You have two options to track your crawl's progress:

#### Option 1: Webhook Notifications (Recommended)

Configure a webhook URL in your [settings page](https://app.olyptik.io/settings/project) to receive real-time status updates. You can choose from two notification types:

- **Crawl Status Changes**: Receive notifications when crawl status changes (succeeded, failed, timed out, etc.)
- **Individual Results**: Receive notifications for each page crawled during the process

You can enable either option or both simultaneously.

#### Option 2: Polling

If you haven't configured webhook notifications, you can poll the crawl status:

<CodeGroup>

```javascript javascript
const crawl = await olyptik.getCrawl(crawlId);
```


```python python
crawl = olyptik.get_crawl(crawlId);
```


```bash curl
curl https://api.olyptik.io/crawls/{crawl_id} \
  -H "Authorization: Bearer your-api-key"
```

</CodeGroup>

### Step 3: Get Crawl Results

Once the crawl is complete, retrieve the results:

<CodeGroup>

```javascript javascript
const page = 1;
const itemsInPage = 50;
const results = await olyptik.getCrawlResults(crawlId, page, itemsInPage);
console.log(results);
```


```python python
page = 1;
itemsInPage = 50;
results = olyptik.get_crawl_results(crawlId, page, itemsInPage);
```


```bash curl
curl https://api.olyptik.io/crawls/{crawl_id}/results?page=1&limit=50 \
  -H "Authorization: Bearer your-api-key"
```

</CodeGroup>
