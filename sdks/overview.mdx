## Common Features

All SDKs provide:

- ✅ **Authentication** - API key management
- ✅ **Crawl Management** - Start, monitor, and abort crawls  
- ✅ **Result Retrieval** - Paginated access to crawl results

## Getting Your API Key

Before using any SDK, you'll need an API key:

1. If your not a memeber, you are welcomed to join: [sign up for Olyptik](https://app.olyptik.io/signup) 
2. Navigate to [Settings → API Keys](https://app.olyptik.io/settings/crawl)
3. Use the key provided
